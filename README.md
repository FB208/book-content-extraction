# 图书内容提取

非通用代码，仅提供思路

演示的书籍是易经解卦，从书中提取格式化的卦象和卦辞

书籍转换文本使用智谱清言
内容抽取用的gemini-2.0-flash，目前只有这个模型支持无限上下文长度
